# ⚔️ Typing RPG - 英语打字战斗精简版

一款基于浏览器的英语单词打字战斗游戏。通过正确拼写单词来攻击怪物，赚取经验和金币，提升等级并解锁更强的技能和词库。

## ✨ 核心特性

- **四级难度系统**：
  - **初学者**：完整显示单词提示。
  - **中级 (默认)**：只显示首尾字母提示。
  - **高级**：仅提示首字母。
  - **宗师**：全盲打，无任何字母提示。
- **丰富的词库支持**：整合了程序员常用词汇、小学必会词汇、大学英语四级（CET-4）高频词汇以及考研核心词汇。
- **RPG 养成系统**：
  - **等级与属性**：击败怪物获得 EXP，提升等级增加 HP 上限并获得更强称号。
  - **技能系统**：金币可购买回血、冰冻（减速）和处决刃等强力技能。
  - **怪物图鉴**：自动记录击败过的不同怪物。
- **极致的打字体验**：
  - **防干扰机制**：特殊设计的输入框，有效抵御系统输入法（IME）和浏览器翻译插件的弹窗干扰。
  - **沉浸式 UI**：采用现代玻璃拟态（Glassmorphism）风格，深色背景更护眼。
  - **音效支持**：内置 Web Audio API 生成的复古 8-bit 打字与打击音效。

## 🚀 快速启动

1.  克隆本仓库到本地：
    ```bash
    git clone https://github.com/SUNkaiyang-m/typing-rpg.git
    ```
2.  双击文件夹中的 `index.html` 即可在浏览器中开始挑战！

## 🛠️ 技术栈

- **前端**：HTML5, Vanilla CSS, Native JavaScript.
- **音频**：Web Audio API.
- **存储**：Local Storage (进度自动保存).
- **反干扰**：IME-Shielding & Translate-Protection Layer.

## 📝 词库来源

- 程序员常用词汇、小学英语。
- 大学英语四级高频词汇（包含音标与中文对照）。
- 考研英语核心词汇。
